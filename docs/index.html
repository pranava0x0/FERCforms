How to use this explorer ▾

A browsable index of FERC Form 1 (electric) audit reports — what FERC's auditors found and recommended. Each card below is one audit.

  • Open a report: click a card to expand its thread — the findings of noncompliance, each with the staff recommendations nested beneath, quoted verbatim from the report.
  • Filter (left rail): by audit type (Financial / Non-financial), function (generation / transmission / distribution), year, or theme — or type in Search. Selections within a filter are OR; across filters, AND.
  • Common patterns (right rail): click a theme to narrow the stream to reports with that issue.
  • Trace the source: each report links to FERC's original PDF via View on eLibrary, and Copy citation grabs a reference.
